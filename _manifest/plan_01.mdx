# Build meta.zip - Privacy-First EXIF Metadata Stripper

Create a professional, open-source web application that removes EXIF metadata from photos entirely in the browser using Rust WebAssembly.

## Project Overview

**Name:** meta.zip  
**Purpose:** Strip sensitive metadata (GPS, camera info, timestamps) from photos before sharing  
**Tech Stack:** Rust (WASM) + React + TypeScript + shadcn/ui + Tailwind  
**Deployment:** Vercel  
**Repository:** GitHub (open source, MIT license)

## Core Functionality

### 1. Photo Processing
- **Batch upload**: Drag-and-drop or click to upload multiple images
- **Supported formats**: JPEG, PNG, HEIC, WebP, TIFF (as many as possible via Rust image libraries)
- **File size limit**: 50MB per image
- **Processing**: Strip ALL EXIF/metadata using Rust WASM
- **Download behavior**:
  - Single image: Direct download
  - 2-4 images: Individual downloads
  - 5+ images: Download as ZIP file

### 2. Metadata Display
Before/after comparison for each image showing:
- **What was removed**: GPS coordinates, camera model, timestamps, software, etc.
- **Visual diff**: Side-by-side or toggle view
- **File size reduction**: Show bytes saved (e.g., "Removed 47KB of metadata")
- Clear indication if no metadata was found

### 3. UI/UX Requirements

**Design Reference:** https://convt.app/ (clean, minimal, intuitive)

**Key Features:**
- **Theme switcher**: Light/Dark/System mode (persistent in localStorage)
- **Drag-and-drop zone**: Large, prominent, with visual feedback
- **Progress indicators**: Per-file progress bars during processing
- **Mobile-first**: Fully responsive, works great on phones
- **Accessibility**: Keyboard navigation, screen reader support, ARIA labels
- **Error handling**: Clear messages for unsupported formats, file too large, etc.

**Layout:**
- Hero section: Clear value prop "Strip metadata from photos. Privately. Instantly."
- Upload zone: Center stage, impossible to miss
- Results section: Expandable cards per image showing before/after metadata
- Footer: GitHub link, "How it works" section

### 4. Privacy & Trust Building

**"How It Works" Section:**
- Explain client-side processing with simple diagrams
- "Your photos never leave your device"
- "No servers, no uploads, no tracking"
- "Open source - verify the code yourself"
- Example metadata screenshot showing privacy risks (GPS coordinates revealing home address)

**Trust signals:**
- Open source badge
- GitHub stars count (configurable to hide/show via environment variable)
- "View source code" prominent link
- No cookies banner needed (actually no cookies!)

## Technical Architecture

### Rust WASM Module

**File:** `rust-wasm/src/lib.rs`

**Functions to expose:**
```rust
// Strip metadata from image bytes, return cleaned bytes
pub fn strip_metadata(image_bytes: Vec<u8>, format: String) -> Result<Vec<u8>, String>

// Extract metadata as JSON for display
pub fn extract_metadata(image_bytes: Vec<u8>) -> Result<String, String>

// Get file size reduction
pub fn calculate_savings(original: Vec<u8>, cleaned: Vec<u8>) -> u32
```

**Rust libraries:**
- `image` - Image processing
- `kamadak-exif` - EXIF parsing/removal
- `wasm-bindgen` - JS interop
- `serde` + `serde_json` - Metadata serialization
- `zip` - Create ZIP archives for batch downloads

**Build setup:**
- `wasm-pack` for building
- Optimize for size (`wasm-opt`)
- Target bundle size: <400KB

## Deployment (Vercel)

### Setup:
1. Push code to GitHub
2. Import project in Vercel
3. Configure:
   - **Build Command**: `pnpm build`
   - **Output Directory**: `dist`
   - **Install Command**: `pnpm install && cd rust-wasm && wasm-pack build --target web`
4. Add environment variables in Vercel dashboard
5. Deploy

### GitHub Actions (`.github/workflows/deploy.yml`):
```yaml
name: Deploy to Vercel

on:
  push:
    branches: [main]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Rust
        uses: actions-rs/toolchain@v1
        with:
          toolchain: stable
          
      - name: Install wasm-pack
        run: curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh
        
      - name: Setup Node
        uses: actions/setup-node@v3
        with:
          node-version: 18
          
      - name: Install pnpm
        run: npm install -g pnpm
        
      - name: Build WASM
        run: cd rust-wasm && wasm-pack build --target web
        
      - name: Install dependencies
        run: pnpm install
        
      - name: Build
        run: pnpm build
        
      - name: Deploy to Vercel
        uses: amondnet/vercel-action@v20
        with:
          vercel-token: ${{ secrets.VERCEL_TOKEN }}
          vercel-org-id: ${{ secrets.ORG_ID }}
          vercel-project-id: ${{ secrets.PROJECT_ID }}
```
